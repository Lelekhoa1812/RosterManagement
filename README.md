Roster Management System
The Roster Management System is an Android application designed to streamline and manage staff scheduling and shift tracking. Developed with Kotlin and Android Studio, this app provides an intuitive interface for administrators to manage shifts, track staff attendance, and handle payment statuses.


A/ Key Features
Staff Management: Easily add and manage staff members.
Shift Scheduling: Create, update, and track shifts with filters for staff names and date ranges.
Attendance Tracking: Record start and end times of shifts and monitor active status.
Payment Status: Update and track payment statuses for each shift.
User Roles: Different access levels for admins and users.
Offline Support: Operates seamlessly without an internet connection, with local database management.


B/ Technologies Used
Kotlin
Android Studio
RecyclerView
Room Database
ViewBinding and DataBinding


C/ Installation
Clone the repository: git clone https://github.com/Lelekhoa1812/RosterManagement.git
Open the project in Android Studio.
Build and run the application on an Android device or emulator.
